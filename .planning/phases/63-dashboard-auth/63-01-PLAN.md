
2. In the requestListener function, add the /api/auth/* route AFTER the CORS preflight check (line ~607) and BEFORE the /mcp block (line ~613). This is CRITICAL — better-auth reads the raw request body; if readRequestBodyWithLimit runs first, better-auth gets empty body and all POST routes fail silently.

   ```typescript
   // ── Phase 63 (AUTH-01, AUTH-02): better-auth routes. DO NOT REORDER.
   // MUST be BEFORE any body-reading code — toNodeHandler reads raw stream.
   if (req.url?.startsWith("/api/auth/")) {
     setCorsHeaders(res);
     return void toNodeHandler(auth)(req, res);
   }
   ```

3. Also add CORS preflight handling for /api/auth/*. Update the existing handleCorsPreflightIfNeeded check OR add a separate guard:

   ```typescript
     res.end();
     return;
   }
   ```
