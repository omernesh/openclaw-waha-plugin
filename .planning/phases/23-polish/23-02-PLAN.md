    **App.tsx — Add isRefreshing + lastRefreshed state and callback plumbing:**

    1. Add two new state variables:
       ```
       const [isRefreshing, setIsRefreshing] = useState(false)
       const [lastRefreshed, setLastRefreshed] = useState<Date | null>(null)
       ```

    2. Create a callback that tabs call to report loading state:
       ```
       const handleTabLoadingChange = useCallback((loading: boolean) => {
         setIsRefreshing(loading)
         if (!loading) setLastRefreshed(new Date())
       }, [])
       ```
       Import `useCallback` from react.

    3. Pass `isRefreshing` and `lastRefreshed` as new props to TabHeader:
       ```
       <TabHeader
         ...existing props...
         isRefreshing={isRefreshing}
         lastRefreshed={lastRefreshed}
       />
       ```

    4. Pass `onLoadingChange={handleTabLoadingChange}` to each tab in renderActiveTab:
       ```
       const props = { selectedSession, refreshKey, onLoadingChange: handleTabLoadingChange }
       ```

    **TabHeader.tsx — Spinner and timestamp display:**

    1. Extend TabHeaderProps interface:
       ```
       isRefreshing?: boolean
       lastRefreshed?: Date | null
       ```

    2. On the refresh Button, add conditional spin class to the RefreshCw icon:
       ```
       <RefreshCw className={cn("h-4 w-4", isRefreshing && "animate-spin")} />
       ```
       Import `cn` from '@/lib/utils'.

    3. Disable the refresh button while isRefreshing is true:
       ```
       <Button ... disabled={isRefreshing} ...>
       ```

    4. After the refresh button, add a timestamp display:
       ```
       {lastRefreshed && (
         <span className="text-xs text-muted-foreground hidden sm:inline">
           {lastRefreshed.toLocaleTimeString()}
         </span>
       )}
       ```

    Each tab already has a `loading` state and `setLoading` calls. Add `onLoadingChange` to each tab's props interface and call it whenever loading state changes.

    Pattern for each tab (DashboardTab, SettingsTab, SessionsTab, QueueTab, ModulesTab, LogTab, DirectoryTab):

    1. Add `onLoadingChange?: (loading: boolean) => void` to the tab's props interface
    2. Add a useEffect that calls onLoadingChange when loading changes:
       ```
       useEffect(() => { onLoadingChange?.(loading) }, [loading, onLoadingChange])
       ```
    3. Do NOT change any existing loading logic — just add the reporting effect
