# Phase 42 Plan 01: Full Regression Testing

## Objective
Validate all v1.14 hardening changes work correctly together and no existing functionality is broken. Run all existing tests, add new tests for v1.14 features, verify clean compile and build.

## Tasks

### Task 1: Run existing tests and fix failures
type="auto"

Run `npx vitest run` and verify all existing tests pass. Fix any failures caused by v1.14 refactors (import changes, signature changes). Verify TypeScript compiles clean with `npx tsc --noEmit`.

**Done criteria:** All existing tests pass, tsc --noEmit clean.

### Task 2: Add new tests for v1.14 features
type="auto"

Add `src/metrics.test.ts` covering:
- collectMetrics() returns valid Prometheus text format with HELP/TYPE lines
- recordApiCall increments counters correctly
- recordHttpRequest tracks route/method/status
- updateQueueStats updates queue depth values
- updateSessionHealth tracks session states

Add `src/v14-features.test.ts` covering:
- Admin auth: requireAdminAuth rejects without token, accepts with valid Bearer token, passes when no token configured
- JID validation: isValidJid accepts valid JIDs (@c.us, @g.us, @lid, @newsletter), rejects malformed
- Config import validation: rejects unknown top-level keys
- SSE client cap: MAX_SSE_CLIENTS = 50, new connections rejected at cap with 503
- Circuit breaker: callWahaApi fast-fails when session is unhealthy
- Graceful shutdown: drain logic waits for in-flight requests

DO NOT test config-io.ts, logger.ts, rate-limiter.ts -- they already have tests.

**Done criteria:** All new tests pass, no regressions.

### Task 3: Verify build
type="auto"

Run `npx tsc --noEmit` for clean compile. Run `npm pack --dry-run` to verify publishable.

**Done criteria:** Clean compile, pack succeeds.

## Verification
