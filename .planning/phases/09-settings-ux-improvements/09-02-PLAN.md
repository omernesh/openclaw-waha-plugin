**Step 3: Initialize tag input AFTER panel DOM element assignment**

In buildGroupPanel(), AFTER the line that assigns the HTML string to the panel element, add the tag input initialization:

```javascript
// UX-03: Initialize tag input for group filter keywords (must be after DOM assignment)
var sfx = groupJid.replace(/[@.]/g, '_');
if (!gfoTagInputs[sfx]) {
  gfoTagInputs[sfx] = createTagInput('gfo-patterns-cp-' + sfx, {
    placeholder: 'hello, help, bot'
  });
}
```

CRITICAL: This must happen AFTER the HTML is in the DOM, not during string construction. The createTagInput function returns null if the container element does not exist. Follow the Phase 8 lazy init pattern.
