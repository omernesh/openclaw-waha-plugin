    - PolicyCache.get(scopeId, differentMtime) returns undefined (stale mtime = miss)

       - `mergeRuleLayers<T extends Record<string, unknown>>(layers: Array<Partial<T> | null | undefined>): Partial<T>` — iterate layers, skip null/undefined.        - This is a pure function with no imports beyond types. Follow the code example from 06-RESEARCH.md exactly.

       - `set(scopeId: string, mtime: number, policy: ResolvedPolicy): void`.
       - `invalidate(scopeId: string): void` — deletes ALL entries matching scopeId prefix (iterate keys).
       - `clear(): void` — full cache clear.
       - Export a module-level singleton: `export const policyCache = new PolicyCache()`.

    - Global manager + appoint_manager => denied (reason: "only owner can appoint/revoke managers")
    - Scope manager + edit_policy at their scope => allowed (reason: "scope_manager")
    - Scope manager + edit_policy at different scope => denied (reason: "not_authorized")
    - Non-manager + edit_policy => denied (reason: "not_authorized")
    - Non-manager + appoint_manager => denied (reason: "only owner can appoint/revoke managers")
  </behavior>
