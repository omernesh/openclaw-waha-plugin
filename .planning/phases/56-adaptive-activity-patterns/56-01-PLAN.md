    **isOffPeak function (private):**
    ```typescript
    function isOffPeak(cfg: CoreConfig, session: string, now: number): boolean {
      const wahaConfig = (cfg as any)?.channels?.waha ?? cfg;
      if (!wahaConfig?.sendGate?.enabled) return true;
      const gateConfig = resolveGateConfig(session, wahaConfig, null);
      if (!gateConfig.enabled) return true;
      const result = checkTimeOfDay(gateConfig, now);
      return !result.allowed; // off-peak = gate would block sends
    }
    ```

    **fetchRecentTimestamps (private async):**
    Paginate getWahaChatMessages in batches of 100. Stop conditions: (a) empty response, (b) oldest message older than 7 days, (c) offset >= 500. WAHA returns Unix seconds — multiply by 1000. Return number[] of timestamps in ms.

    **computePeakWindow (exported):**
    Per research — hour histogram via Intl.DateTimeFormat.formatToParts. Top 60% of hours by volume. Return null if < 20 timestamps. Include code comment about bimodal activity producing wide windows (intentionally permissive).
    Handle midnight normalization: if hourPart.value is "24", treat as 0 (Intl quirk).

    **runScanBatch (private async):**
    - Get chats needing rescan via dirDb.getChatsNeedingRescan(accountId, 7*86400000, 7*86400000)
    - Get cursor from scanCursors map (default 0)
    - Process BATCH_SIZE=10 chats starting at cursor with 500ms sleep between each
    - For each chat: fetchRecentTimestamps -> computePeakWindow -> if result, upsertActivityProfile
    - Update cursor. If batch exhausted (no more chats), reset cursor to 0
    - Check abortSignal.aborted between each chat scan

    **tick (private async):**
    - If abortSignal.aborted, return
    - If NOT isOffPeak, log debug, scheduleNext 30 minutes, return (ADAPT-03)
    - Set state.status = "running"
    - Try runScanBatch, catch and log errors
    - Set state.status = "idle"
    - If cursor was reset to 0 (full pass done): scheduleNext 7 days (ADAPT-02 weekly rescan)
    - Else: scheduleNext 5 minutes (continue batch in next tick)

    **startActivityScanner:**
    - Create ScannerState, store in scannerStates map
    - Register abortSignal listener to clean up maps
    - Schedule first tick after 30s delay (.unref())
    - Return state
