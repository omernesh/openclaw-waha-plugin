**Part D: createTagInput factory function (add after normalizeTags)**

Function signature: `function createTagInput(containerId, opts)`
- opts defaults to {}
- Resolve containerId to element
- Maintain closure var `tags = []`
- Create div.ti-wrap container
- Create input.ti-input with placeholder from opts.placeholder or "Type JID or phone, press Enter"
- Append input to wrap, clear container, append wrap

Internal `renderTags()`:
- Remove all .ti-tag elements from wrap (querySelectorAll, removeChild loop)
- For each tag in array: create span.ti-tag, set its text to tag value + space, create span.ti-remove with text "\u00d7" (multiplication sign) and aria-label "Remove {tag}", add click listener (with closure for index) that splices tag and re-renders
- Insert each pill before the input element using wrap.insertBefore(pill, input)

Internal `addTag(val)`:
- Trim val, skip if empty
- Skip if already in tags array (indexOf check for dedup)
- Push to tags, clear input.value, call renderTags()

Event listeners:
- input keydown: if Enter/comma/space/Tab and input has trimmed value, preventDefault, split with normalizeTags, add each. If Backspace and input empty and tags has items, pop last tag and renderTags.
- input paste: setTimeout 0 then normalizeTags on input.value, addTag each
- wrap click: focus input
- input focus: add ti-focused class to wrap
- input blur: remove ti-focused class, if input has trimmed value call addTag

Return object: `{ getValue: function() { return tags.slice(); }, setValue: function(arr) { tags = (arr || []).slice(); renderTags(); } }`
