Common patterns:

Send instruction:
```bash
ssh omer@100.114.126.43 'curl -s -X POST http://127.0.0.1:3004/api/sendText \
  -H "Content-Type: application/json" \
  -H "X-Api-Key: XcTCX9cn84LE/uMm3SnHEvm0giwtNnHBmGR7OGeAOpA=" \
  -d "{\"chatId\":\"120363421825201386@g.us\",\"text\":\"INSTRUCTION\",\"session\":\"3cf11776_omer\"}"'
```

Check agent reply:
```bash
ssh omer@100.114.126.43 'curl -s "http://127.0.0.1:3004/api/3cf11776_logan/chats/120363421825201386@g.us/messages?limit=5&downloadMedia=false" \
  -H "X-Api-Key: XcTCX9cn84LE/uMm3SnHEvm0giwtNnHBmGR7OGeAOpA="'
```

TST-01 Add/Remove participant:
- Send: "Add Michael Greenberg (972556839823@c.us) to this group"
- Wait 30s. Verify via participants API:
  ssh omer@100.114.126.43 'curl -s "http://127.0.0.1:3004/api/3cf11776_logan/groups/120363421825201386@g.us/participants" -H "X-Api-Key: XcTCX9cn84LE/uMm3SnHEvm0giwtNnHBmGR7OGeAOpA="'
- Confirm 972556839823@c.us in list.
- Then send: "Remove Michael Greenberg from this group"
- Wait 30s. Verify he is gone.

TST-02 Promote/Demote admin:
- Add Michael back: "Add 972556839823@c.us to this group"

- Send: "Demote Michael Greenberg from admin in this group"
- Wait 30s. Verify isAdmin=False.
- Remove Michael: "Remove 972556839823@c.us from this group"

TST-03 Group subject and description:

- Send: "Set this group's description to 'Testing v1.19 action exposure'"

- Restore: "Change this group's name back to 'sammie test group'"

TST-04 Group picture set/delete:

- Send: "Delete this group's picture"
- Wait 30s. Check agent reply for success.
