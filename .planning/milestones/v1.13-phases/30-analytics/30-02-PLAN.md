**Create src/admin/src/components/tabs/AnalyticsTab.tsx:**

Follow the tab pattern (QueueTab/DashboardTab). The component should:

1. **Props**: `{ selectedSession, refreshKey, onLoadingChange }` (standard tab props)
2. **State**: `data: AnalyticsResponse | null`, `loading: boolean`, `range: string` (default '24h')
3. **Data fetch**: useEffect on [refreshKey, range] — call `api.getAnalytics(range)`, set data, handle loading.
4. **Report loading** to parent via `onLoadingChange` (same pattern as other tabs).

**Layout** (use Card components from shadcn):

A. **Range selector** at top — shadcn Select or simple button group with options: 1h, 6h, 24h, 7d, 30d. Active range highlighted.

B. **Summary cards row** — 4 small cards in a grid (grid-cols-2 lg:grid-cols-4):
   - Total Messages (summary.total)
   - Inbound (summary.inbound) with green accent
   - Outbound (summary.outbound) with blue accent
   - Avg Response (summary.avg_duration_ms formatted as "Xms" or "X.Xs")

C. **Messages chart** — recharts BarChart inside a Card:
   - Stacked bars: inbound (green-500) + outbound (blue-500)
   - X-axis: period labels (format based on groupBy — hour: "HH:00", day: "MM/DD", minute: "HH:MM")
   - Y-axis: count
   - Tooltip showing period, inbound, outbound, errors
   - ResponsiveContainer width="100%" height={300}
   - Use recharts: `BarChart, Bar, XAxis, YAxis, CartesianGrid, Tooltip, Legend, ResponsiveContainer`

D. **Response time chart** — recharts LineChart inside a Card:
   - Line for avg_duration_ms (amber-500)
   - Same X-axis as messages chart
   - Y-axis: ms
   - ResponsiveContainer width="100%" height={250}
   - Use recharts: `LineChart, Line, XAxis, YAxis, CartesianGrid, Tooltip, ResponsiveContainer`

E. **Top chats table** — inside a Card:
   - Simple table with columns: Chat ID, Total, Inbound, Outbound
   - Truncate chat_id display (show first 20 chars + "...")
   - Show "No data" if topChats is empty

F. **Empty state**: If summary.total === 0, show a centered "No analytics data yet" message.
G. **Loading state**: Show Skeleton components (same pattern as other tabs).

Use `hsl(var(--chart-1))` through `--chart-5` CSS variables for chart colors (shadcn chart theming), falling back to direct Tailwind colors if those vars aren't available. Use the simpler approach: direct fill/stroke colors like `#22c55e` (green) and `#3b82f6` (blue).
