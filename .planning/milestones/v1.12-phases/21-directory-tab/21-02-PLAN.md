**2. Create `ContactSettingsSheet.tsx`** — Side panel for editing contact DM settings.

Props:
```typescript
interface ContactSettingsSheetProps {
  jid: string | null            // null = closed
  displayName: string | null
  dmSettings?: ContactDmSettings
  allowedDm: boolean
  onClose: () => void
  onSaved: () => void           // called after successful save to trigger data refresh
}
```

Implementation:
- Use shadcn `Sheet` component (SheetContent, SheetHeader, SheetTitle, SheetDescription)
- `open={!!jid}` — Sheet is open when jid is set
- `onOpenChange` — calls onClose when user closes via X or overlay click
- **Form fields** (all controlled state, initialized from props on open):
  - **Allow DM toggle**: `Switch` — calls `api.toggleAllowDm(jid, { allowed })` immediately on toggle, shows toast
  - **Mode**: `Select` with options "Active" (value `active`) and "Listen Only" (value `listen_only`)
  - **Mention Only**: `Switch`
  - **Can Initiate Override**: `Select` with options "Default" (`default`), "Allow" (`allow`), "Block" (`block`)
  - **Custom Keywords**: `TagInput` in freeform mode
    - **CRITICAL**: `customKeywords` comes from API as a comma-separated string. Split on load: `(dmSettings?.customKeywords ?? '').split(',').filter(Boolean)`. Join on save: `keywords.join(',')`
  - **TTL Access**: Display current expiry if `expiresAt` is set. Buttons: "Grant 24h", "Grant 7d", "Revoke". Call `api.setDirectoryTtl(jid, { expiresAt })`.
- **Save button**: Calls `api.updateDirectorySettings(jid, { mode, mentionOnly, customKeywords: keywords.join(','), canInitiateOverride })`. On success: `toast.success('Settings saved')` via sonner. **DO NOT close the sheet** after save — per locked decision. Call `onSaved()` to trigger parent refresh.
- Initialize form state from props using `useEffect` that watches `jid` — when jid changes, reset all form fields to match props.
